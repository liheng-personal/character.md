# Examples

This directory contains example CHARACTER.md projects at different levels of complexity.

## [minimal/](minimal/)

A single-file CHARACTER.md for a TRPG-style character. Everything — dispositions, knowledges, and experiences — lives in one file. Good for small characters, quick prototypes, or getting a feel for the format.

## full/ (coming soon)

A complete CHARACTER.md project with the full directory structure: main file plus dispositions/, knowledges/, and experiences/ folders. Demonstrates how a complex character's memory scales across multiple files.
